# BasePart.lua

## GetBasePartBounds
Возвращает границы BasePart в виде таблицы с минимальными и максимальными значениями.
### Параметры:
- basePart: BasePart, для которого нужно получить границы.
### Возвращает:
- Min: Vector3, минимальные значения границ.
- Max: Vector3, максимальные значения границ.

# CameraCoordinate.luau

## GetPositionAndAnglesFromCamera
Возвращает позицию и углы камеры с учетом смещения.
### Параметры:
- camera: Camera, камера, относительно которой нужно получить позицию и углы.
- positionOffset: Vector3?, смещение позиции.
- rotationOffset: Vector3?, смещение углов.
### Возвращает:
- Position: Vector3, позиция камеры с учетом смещения.
- Rotation: Vector3, углы камеры с учетом смещения.
- Degrees: Vector3, углы камеры в градусах.
- Look: Vector3, вектор направления взгляда камеры.

# CharacterFinder.luau

## GetHumanoidByPart
Возвращает Humanoid, связанный с переданной частью.
### Параметры:
- part: BasePart, часть, для которой нужно найти связанный Humanoid.
- enableExactSearchMethod: boolean?, включает точный метод поиска.
- name: string?, имя для поиска.
### Возвращает:
- Humanoid?, связанный с частью Humanoid.

## GetPlayerByPart
Возвращает игрока, связанного с переданной частью.
### Параметры:
- part: BasePart, часть, для которой нужно найти связанного игрока.
- enableExactSearchMethod: boolean?, включает точный метод поиска.
- name: string?, имя для поиска.
### Возвращает:
- Player?, связанный с частью игрок.

# Coordinate.luau

## GetPositionOnNormal
Возвращает позицию на нормали относительно переданного CFrame.
### Параметры:
- originCframe: CFrame, относительно которого нужно получить позицию.
- originSize: Vector3, размер объекта.
- normalId: Enum.NormalId, нормаль, относительно которой нужно получить позицию.
- positionOffset: Vector3?, смещение позиции.
### Возвращает:
- Vector3, позиция на нормали.

## GetNormalRelativeToPosition
Возвращает нормаль, ближайшую к переданной позиции.
### Параметры:
- originCFrame: CFrame, относительно которого нужно получить нормаль.
- originSize: Vector3, размер объекта.
- position: Vector3, позиция, относительно которой нужно получить нормаль.
### Возвращает:
- Enum.NormalId?, нормаль, ближайшая к переданной позиции.

# Physics.luau

## ApplyForce
Применяет силу к переданной части.
### Параметры:
- basePart: BasePart, часть, к которой нужно применить силу.
- velocity: Vector3, вектор скорости.
- duration: number?, продолжительность применения силы.

## ApplyForceFromLook
Применяет силу к переданной части в направлении взгляда другой части.
### Параметры:
- basePart: BasePart, часть, к которой нужно применить силу.
- basePartLook: BasePart, часть, направление взгляда которой определяет направление силы.
- velocity: Vector3, вектор скорости.
- duration: number?, продолжительность применения силы.

# Table.luau

## GetKeyByIndex
Возвращает ключ и значение по переданному индексу.
### Параметры:
- _table: {[string]: any}, таблица, в которой нужно получить ключ и значение.
- index: number, индекс, по которому нужно получить ключ и значение.
### Возвращает:
- {}?, ключ и значение по переданному индексу.

## GetKeyIndexByName
Возвращает индекс по переданному имени ключа.
### Параметры:
- _table: {}, таблица, в которой нужно получить индекс.
- name: string, имя ключа, по которому нужно получить индекс.
### Возвращает:
- number?, индекс по переданному имени ключа.

## GetKeyIndexByValue
Возвращает индекс по переданному значению.
### Параметры:
- _table: {}, таблица, в которой нужно получить индекс.
- value: a, значение, по которому нужно получить индекс.
### Возвращает:
- number?, индекс по переданному значению.

## ReplaceKeys
Заменяет ключи в переданной таблице на новые из переданной таблицы сопоставлений.
### Параметры:
- _table: {}, таблица, в которой нужно заменить ключи.
- keyMappings: {}, таблица сопоставлений старых и новых ключей.

## UpdateExistingKeys
Обновляет существующие ключи в переданной таблице на новые из переданной таблицы.
### Параметры:
- _table: {}, таблица, в которой нужно обновить ключи.
- fromTable: {}, таблица с новыми значениями для ключей.
